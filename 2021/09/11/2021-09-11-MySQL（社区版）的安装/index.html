3.11配置完成，下面是环境变量设置

3.11.添加环境变量

在c盘下 找到这个路径C:\Program Files\MySQL\MySQL Server 8.0\bin，复制下来–>此电脑–> 属性–> 高级系统设置–>环境变量—>用户变量–> path–> 编辑–>新建–>粘贴路径–>一路确定退出

3.12.在命令提示行启动mysql和更改密码

打开命令提示行（win+R输入cmd）

输入命令mysql -u root -p 回车

输入密码后回车（做过3.10步骤的不用输入密码，因为你的密码为空，直接回车就好看3.13）

3.13修改密码

alter user ‘root‘@’localhost’ identified by ‘密码’
